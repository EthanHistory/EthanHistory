# Priority Queue

a **priority queue** is an [abstract data-type](https://en.wikipedia.org/wiki/Abstract\_data\_type) similar to a regular [queue](https://en.wikipedia.org/wiki/Queue\_\(abstract\_data\_type\)) or [stack](https://en.wikipedia.org/wiki/Stack\_\(abstract\_data\_type\)) data structure in which each element additionally has a "priority" associated with it.
